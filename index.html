Sobre Mim

Sou estudante de Análise e Desenvolvimento de Sistemas no SENAI, com foco em desenvolvimento web full stack e construção de aplicações modernas utilizando JavaScript e Node.js.

Atualmente desenvolvo projetos próprios com HTML, CSS, JavaScript ES6+, Node.js, Express.js e MySQL, aplicando conceitos como arquitetura modular, consumo de APIs, manipulação de DOM, navegação dinâmica, persistência de dados e integração entre frontend e backend.

Tenho experiência prática no desenvolvimento de aplicações SPA (Single Page Application), incluindo sistemas com carregamento dinâmico de páginas, estrutura modular, filtros avançados, gráficos interativos, organização de serviços compartilhados e construção de APIs REST.

Também possuo conhecimentos sólidos em banco de dados relacionais, incluindo modelagem de dados, relacionamentos, normalização, consultas SQL, manipulação de dados, filtros dinâmicos e estruturação de sistemas utilizando MySQL.

Atualmente estou aprofundando meus estudos em tecnologias modernas do ecossistema JavaScript, como React, TypeScript, Next.js e NestJS, além de conhecimentos em Java, Spring Boot e desenvolvimento backend estruturado.

Utilizo Git e GitHub no fluxo de desenvolvimento, com foco em organização de código, versionamento, boas práticas e evolução contínua na construção de aplicações cada vez mais robustas e escaláveis.

Além do desenvolvimento web, também possuo conhecimentos complementares em redes, infraestrutura e hardware, ampliando minha visão técnica sobre sistemas e tecnologia.
